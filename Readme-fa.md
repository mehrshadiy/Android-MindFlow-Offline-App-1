# MindFlow (Android)
README (Offline Version)
MindFlow (اندروید - نسخه آفلاین)

MindFlow یک برنامه پایش سلامت روان است که با **Kotlin + Jetpack Compose** و با استفاده از **MVVM + Clean Architecture** ساخته شده است. این نسخه تمام داده‌ها را به صورت محلی با استفاده از **Room Database** و کاملاً آفلاین ذخیره می‌کند.

# ویژگی‌ها 
- ثبت روزانه خلق و خو (عالی → خیلی بد) با یادداشت‌های اختیاری
- فهرست خلق و خوهای ثبت شده
- نمودارهای ساده جایگزین شده با فهرست خلق و خو
- مدیریت تمرین‌های آرامش‌بخش
- حالت مدیریت درون برنامه‌ای (محافظت شده با پین) برای مدیریت تمرین‌ها و مشاهده داده‌های کاربر
- کاملاً آفلاین، بدون نیاز به فایربیس

# معماری 
- MVVM
- الگوی مخزن
- فضای پایگاه داده محلی (خلق و خو و تمرین‌ها)
- جت‌پک کامپوز برای رابط کاربری

# پایگاه داده 
- **جدول خلق و خو**: شناسه، نوع خلق و خو، یادداشت، تاریخ
- **جدول تمرین**: شناسه، عنوان، توضیحات، نوع
